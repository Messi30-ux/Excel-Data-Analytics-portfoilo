**Superstore Sales & Profitability Dashboard (Excel)**

**Project Overview:** This project analyzes retail sales data to uncover insights about revenue, profit, discount impact, and regional performance. An interactive Excel dashboard was built to help stakeholders make data-driven decisions.

**Tools Used:**	Microsoft Excel, Pivot Tables, Pivot Charts, Slicers, Data Cleaning Techniques

**Business Questions Answered:** 1.	Which region generates the most revenue? 2.	How do discounts affect profitability? 3.	Which product categories are most profitable? 4.	What products are causing losses? 5.	How have sales changed over time?

**Key Insights:** •	Total Sales: $2.29M •	Total Profit: $286K •	Profit margins are heavily affected by discounting. •	West region drives the highest revenue.
•	Some sub-categories are unprofitable and require review.
